Stealth Technology

The project How Radar Works talks about the basic principles of a radar system. The idea is for the radar antenna to send out a burst of radio energy, which is then reflected back by any object it happens to encounter. The radar antenna measures the time it takes for the reflection to arrive, and with that information can tell how far away the object is.
